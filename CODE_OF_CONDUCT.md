# Code of Conduct

This project follows the spirit of the Contributor Covenant.

## Our Standards

- Be respectful and constructive.
- Focus on the code and the issue.
- Avoid harassment, insults, and personal attacks.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.

## Scope

This code applies to public project spaces, including issues, pull requests, and discussions.

## Reporting

If you experience behavior that violates this code of conduct, contact the maintainers through the repository issue tracker or the security contact listed in SECURITY.md.
